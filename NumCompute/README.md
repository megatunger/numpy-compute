# NumCompute

`NumCompute` is a python library dedicated for educational purpose. It's built upon pure python and numpy.

## Installation

### Dependencies

NumCompute requires:
- Python (>= 3.11)
- Numpy (>= 1.26.4)

### Installation
Make sure you already have a ready python environment with executable pip. To install NumCompute, run:


    git clone https://github.com/danh-real/SSE-ProgAIML.git
    cd NumCompute
    pip install -e .

## Quickstart

To use NumCompute in your code, import it as follow:

    import numcompute as nc

Or you can import the specific function you need:

    from numcompute.io import load_csv

NumCompute are arranged into modules with corresponding functionalities:
- io
- metrics
- optim
- pipeline
- preprocessing
- rank
- sort_search
- stats
- utils

For further demonstration, please refer to `NumCompute/demo/quickstart.ipynb`
